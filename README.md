🌐 Personal Portfolio Website

A modern, responsive, and professional portfolio website developed using HTML5 and CSS3 to showcase academic achievements, technical skills, projects, leadership experience, and professional contact information. The portfolio is designed with a sleek developer-inspired user interface featuring a dark theme, glassmorphism effects, smooth navigation, responsive layouts, and interactive components to provide an engaging user experience.

🚀 Features
Modern dark-themed UI with futuristic design aesthetics
Fully responsive layout optimized for desktop, tablet, and mobile devices
Sticky navigation bar with smooth scrolling between sections
Professional hero section displaying personal information and social links
Detailed About Me section highlighting career objectives and technical interests
Education timeline with academic performance overview
Technical skills categorized into AI Engineering, Programming, Database Management, and Leadership skills
Project showcase section featuring:
AI Campus Assistant (RAG-Based)
Student Management System
Personal Portfolio Website
Leadership and achievements section highlighting organizational roles and accomplishments
Interactive contact section with direct links for phone, email, LinkedIn, and GitHub
Clean card-based design with hover effects and modern typography
Optimized structure for easy customization and future scalability
